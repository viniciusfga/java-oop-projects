# 🏥 Sistema Hospitalar - README

## 📌 Descrição

Este projeto representa um **Sistema de Gestão Hospitalar** modelado utilizando **Diagrama de Classes (UML)**. O objetivo é organizar e estruturar as principais entidades e relações envolvidas no funcionamento de um hospital, como pacientes, médicos, consultas e prontuários.

O sistema segue os princípios de **Programação Orientada a Objetos (POO)**, com uso de herança, associação e encapsulamento.

---

## 🎯 Objetivos

* Modelar um sistema hospitalar completo
* Aplicar conceitos de orientação a objetos
* Representar relações do mundo real em software
* Servir como base para implementação futura (Java, Python, etc.)

---

## 🚀 Possíveis Melhorias
